# UI checklist

Before considering the task done, verify:

- Page has one clear primary action
- No unnecessary nested cards or wrapper div soup
- Existing components reused where possible
- Loading state exists
- Empty state exists
- Error state exists where applicable
- Form errors are visible and understandable
- Mobile layout does not overflow
- Desktop layout does not look sparse or chaotic
- Buttons and inputs have sensible heights
- Numeric/tabular content is aligned properly
- Headings and section spacing feel consistent
- No new visual tokens were introduced without reason
