## Rules

- Use design-system icon exports (`@/design-system/icons`) so the UI stays consistent.
- Follow icon sizing tokens (`--icon-size-sm`, `--icon-size-md`) instead of arbitrary pixel values.
- Prefer icons with clear semantics (use labels + accessible text in controls).
