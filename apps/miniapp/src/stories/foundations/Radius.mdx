## Why it exists

Consistent border radius creates visual harmony and defines surface elevation. Use tokens instead of arbitrary values.

## Rules

- Use `--radius-*` tokens for all rounded corners
- Do not hardcode border-radius values
- Match radius to component size (small components = smaller radius)

## Implementation notes

- Tokens: `--radius-sm`, `--radius-md`, `--radius-lg`, `--radius-xl`, `--radius-full`
- Use `--radius-full` for pills and avatars
