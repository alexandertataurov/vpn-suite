# Doc Naming Conventions

All markdown files in `docs/` use **kebab-case** for consistency and URL-friendliness.

| Pattern | Example |
|---------|---------|
| Specs | `operator-dashboard-spec.md` |
| Audits | `admin-servers-audit.md` |
| Runbooks | `amneziawg-obfuscation-runbook.md` |
| Contracts | `config-generation-contract.md` |
| Guides | `agent-mode-one-server.md` |

When adding new docs, follow kebab-case. Prefer lowercase with hyphens; avoid UPPER_SNAKE or PascalCase.
