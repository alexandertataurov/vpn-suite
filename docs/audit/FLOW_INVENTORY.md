## Miniapp Flow Inventory


| Flow                          | Business goal                                                     | Entry points                                            | Primary APIs                                                                                                                                                           | Page-model / owner                                 |
| ----------------------------- | ----------------------------------------------------------------- | ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------- |
| App open                      | Bring user to correct shell (home vs onboarding vs restore)       | Telegram miniapp launch                                 | `/webapp/auth`, `/webapp/me`, `/webapp/onboarding/state`                                                                                                               | `useBootstrapMachine`                              |
| Home load                     | Show connection, subscription, and quick actions                  | `/`                                                     | `/webapp/me`, `/webapp/servers`, `/webapp/usage`                                                                                                                       | `useHomePageModel`                                 |
| Onboarding                    | Move first-time user to plan/devices with synced onboarding state | `/onboarding` redirect from bootstrap                   | `/webapp/me`, `/webapp/onboarding/state`                                                                                                                               | `useBootstrapMachine` + `OnboardingPage`           |
| Plan selection                | Choose/upgrade/renew subscription tier                            | `/plan`, CTAs from home, devices, referral              | `/webapp/plans`, `/webapp/usage`, `/webapp/payments/history`                                                                                                           | `usePlanPageModel`                                 |
| Checkout / payment            | Activate or renew plan with Stars or free activation              | `/plan/checkout/:planId`                                | `/webapp/plans`, `/webapp/payments/create-invoice`, `/webapp/payments/{id}/status`                                                                                     | `useCheckoutPageModel`                             |
| Subscription restore          | Restore expired/grace subscription and route to next step         | `/restore-access`, CTAs from plan/support               | `/webapp/subscription/restore`, `/webapp/me`                                                                                                                           | `useRestoreAccessPageModel`                        |
| Device issue                  | Issue first/additional device config and show config              | `/devices`, `/devices/issue`, CTAs from plan/onboarding | `/webapp/me`, `/webapp/plans`, `/webapp/devices/issue`                                                                                                                 | `useDevicesPageModel`                              |
| Device replace/revoke/confirm | Manage existing device lifecycle                                  | `/devices`                                              | `/webapp/devices/{id}/replace-with-new`, `/webapp/devices/{id}/revoke`, `/webapp/devices/{id}/confirm-connected`                                                       | `useDevicesPageModel`, `useConnectStatusPageModel` |
| Server selection              | Choose routing mode and preferred server                          | `/servers`, link from devices                           | `/webapp/servers`, `/webapp/servers/select`                                                                                                                            | `useServerSelectionPageModel`                      |
| Referral attach               | Attach incoming referral code to session                          | App launch with `start_param` / `?ref=`                 | `/webapp/referral/attach`                                                                                                                                              | `useReferralAttach`                                |
| Referral screen               | Show referral link, stats, and upsell                             | `/referral`, link from settings/home                    | `/webapp/referral/my-link`, `/webapp/referral/stats`, `/webapp/plans`                                                                                                  | `useReferralPageModel`                             |
| Support hub                   | Route user to restore, plan, devices, or external support         | `/support`, link from home                              | `/health/ready`, `/webapp/me`                                                                                                                                          | `useSupportPageModel`                              |
| Settings / account            | Profile, locale, subscription pause/cancel, revoke configs        | `/settings`                                             | `/webapp/me`, `/webapp/subscription/offers`, `/webapp/subscription/pause`, `/webapp/subscription/resume`, `/webapp/subscription/cancel`, `/webapp/devices/{id}/revoke` | `useSettingsPageModel`                             |
