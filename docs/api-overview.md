# API overview

Base URL: `https://$PUBLIC_DOMAIN` (or `http://localhost:8000` in dev).

Agent (mTLS): `https://$PUBLIC_DOMAIN:8443/api/v1/agent/*` (requires client cert + `X-Agent-Token`).

OpenAPI schema: `openapi/openapi.yaml` (generated by `./manage.sh openapi`).

## S8-4 Auto-block (background)

When `LIMITS_AUTO_BLOCK_ENABLED=true` and `LIMITS_CHECK_INTERVAL_SECONDS>0`, a background task periodically fetches per-peer metrics from discovered AWG runtime nodes via `docker exec ... wg show <iface> dump` (traffic counters per peer). If a device's traffic exceeds the server's `traffic_limit_gb`, the peer is blocked via runtime peer control and an audit entry is written (`admin_id=system`, action `auto_block`).
