# Checkbox

## Purpose
Boolean toggle with label.

**When to use:** Binary selections, table row selection.

**When not to use:** Single-choice among many options (use Radio).

## API (Props)

| Prop | Type |
| --- | --- |
| label | `string` |
| nativeProps | `Omit<InputHTMLAttributes<HTMLInputElement>, "type">` |

## Examples

```tsx
<Checkbox label="Enable alerts" checked={enabled} onChange={...} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
