# Divider

## Purpose
Separator between sections or items.

**When to use:** Split grouped content or rows.

**When not to use:** Pure decoration or heavy visual clutter.

## API (Props)

| Prop | Type |
| --- | --- |
| orientation? | `"horizontal"` | "vertical" |
| tone? | `"subtle"` | "default" |
| nativeProps | `HTMLAttributes<HTMLDivElement>` |

## Examples

```tsx
<Divider orientation="horizontal" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
