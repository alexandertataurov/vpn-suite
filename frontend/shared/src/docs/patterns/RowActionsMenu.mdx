# RowActionsMenu

## Purpose
Contextual actions menu for table rows.

**When to use:** Server or device rows with multiple actions.

**When not to use:** Single primary action (use inline button).

## API (Props)

Use `RowActionsMenu` with row-specific handlers and labels.

## Examples

```tsx
<RowActionsMenu
  serverId="srv-123"
  onRestart={() => {}}
  onDelete={() => {}}
/>
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Typography: `--text-*`

## Anti-patterns
- Hiding essential actions only inside the menu.
- Inline styles or arbitrary spacing.

## Accessibility
- Ensure menu items are keyboard navigable.
