# CopyButton

## Purpose
Button that copies a value to clipboard.

**When to use:** Copy tokens, IDs, commands.

**When not to use:** Primary actions unrelated to copy.

## API (Props)

| Prop | Type |
| --- | --- |
| value | `string` |
| label? | `string` |
| copiedMessage? | `string` |
| onCopy? | `() => void` |

## Examples

```tsx
<CopyButton value={token} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
