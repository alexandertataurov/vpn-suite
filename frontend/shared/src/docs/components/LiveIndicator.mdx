# LiveIndicator

## Purpose
Live/paused/reconnecting status indicator.

**When to use:** Stream status hints.

**When not to use:** Critical status display (use StatusBadge).

## API (Props)

| Prop | Type |
| --- | --- |
| status | `"live"` | "paused" | "reconnecting" | "error" |

## Examples

```tsx
<LiveIndicator status="live" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
