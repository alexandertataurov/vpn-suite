# CodeText

## Purpose
Inline code text styling.

**When to use:** Identifiers, short command snippets.

**When not to use:** Multi-line blocks.

## API (Props)

| Prop | Type |
| --- | --- |
| children | `ReactNode` |
| className? | `string` |

## Examples

```tsx
<CodeText>vpnctl</CodeText>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
