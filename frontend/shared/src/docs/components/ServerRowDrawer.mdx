# ServerRowDrawer

## Purpose
Detail drawer for a server row, including overview, IPs, telemetry, and activity.

**When to use:** Inline drill-down from a server table row.

**When not to use:** Full-page server details (use a dedicated page).

## API (Props)

| Prop | Type |
| --- | --- |
| server | `ServerOut \| null` |
| onClose | `() => void` |
| peerCount? | `number` |
| onRestart? | `(server: ServerOut) => void` |
| telemetrySnapshot? | `ServerSnapshotSummaryEntry \| null` |

## Examples

```tsx
<ServerRowDrawer
  server={server}
  onClose={() => setOpen(false)}
  peerCount={12}
  telemetrySnapshot={snapshot}
/>
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`
- Typography: `--text-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Rendering without a `server` prop.

## Accessibility
- Drawer provides focus trapping and escape support.
- Ensure tab order follows content sections.
