# Do / Don't

## Do
- Use `PrimitiveButton` for actions.
- Use `PrimitiveField` to wrap inputs with labels and errors.
- Use `PrimitiveStack` and `PrimitiveGrid` for layout.
- Use `PrimitiveText` and `PrimitiveHeading` for typography.

## Don't
- Don't add new variants without justification.
- Don't use raw hex/px values in component CSS.
- Don't add inline styles in primitives or composites.
