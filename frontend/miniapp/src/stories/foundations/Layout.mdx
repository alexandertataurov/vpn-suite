## Why it exists

Layout tokens define viewport breakpoints, container widths, and responsive behavior. Use them for media queries and JS layout logic.

## Rules

- Use `--bp-*` or `--adaptive-bp-*` for breakpoints
- Mobile-first: base styles for small screens, `min-width` for larger
- Containers: `--max-width-container-md`, `--max-width-container-lg`

## Implementation notes

- Breakpoints: `--bp-sm` (480px), `--bp-md` (768px), `--bp-lg` (1024px), `--bp-xl` (1280px)
- Adaptive: `--adaptive-bp-xs` (40rem), `--adaptive-bp-sm` (64rem), etc.
- Use `useBreakpoint()` from design-system for JS breakpoint checks
