# Color Semantics

Use these tokens in components. They map to primitives and adapt to theme (dark/light/dim).

> **Theme switcher (toolbar):** Use dark | light | dim to preview semantics in each theme. Swatches update live.

## Background & Surface

| Token | `Usage` | Mapping |
| ------- | `-------` | --------- |
| `--color-background-primary` | `Page, chrome` | gray-950 |
| `--color-background-secondary` | `Cards, raised surfaces` | gray-900 |
| `--color-background-tertiary` | `Nested surfaces` | gray-800 |
