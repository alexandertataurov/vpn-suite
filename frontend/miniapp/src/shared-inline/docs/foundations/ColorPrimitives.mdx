# Color Primitives

Raw color ramps. **Do not use in components.** Use semantic tokens instead.

> **Theme:** Switch dark/light/dim in the toolbar to see semantic tokens in context. Primitives (below) are theme-independent.

## Gray scale

<div className="sb-row sb-mb-8">
  {[50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950].map((n) => (
    <TokenSwatch key={n} cssVar={`--color-gray-${n}`} usage={`gray-${n}`} heightToken="--spacing-12" showBorder={n >= 400} />
  ))}
</div>
