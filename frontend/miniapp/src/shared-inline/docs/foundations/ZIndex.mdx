# Elevation & Z-Index

Operator UI uses a fixed elevation and z-index scale. Use CSS variables in stylesheets and constants in JS.

## Shadow Scale (Elevation Ladder)

<div className="sb-shadow-grid">
  {["xs", "sm", "md", "lg", "xl", "2xl"].map((s) => (
    <div key={s} className="sb-shadow-item" data-shadow={s}>
      <div className="sb-shadow-tile" aria-hidden />
      <code className="sb-code">shadow-{s}</code>
    </div>
  ))}
</div>

## Tokens

- `--z-dropdown`
- `--z-overlay`
- `--z-modal`
- `--z-toast`

## Usage

- CSS: `z-index: var(--z-modal);`
- JS: import `Z_MODAL` from `@vpn-suite/shared/theme`

## Do

- Keep overlays above interactive content.
- Use the next higher token for stacked overlays (dropdown inside modal uses `--z-dropdown` + `--z-modal`).

## Don't

- Hardcode numeric z-index values.
- Introduce new z-layers without updating tokens.
