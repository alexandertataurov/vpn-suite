# TableCell

## Purpose
Table cell primitive for alignment, truncation, and numeric styling.

**When to use:** Table body cells.

**When not to use:** Non-tabular layouts.

## API (Props)

| Prop | Type |
| --- | --- |
| align? | `"left" \| "center" \| "right"` |
| truncate? | `boolean` |
| numeric? | `boolean` |
| mono? | `boolean` |
| nativeProps | `TdHTMLAttributes<HTMLTableCellElement>` |

## Examples

```tsx
<TableCell numeric align="right">42</TableCell>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using table cells outside a table.

## Accessibility
- Use `<th>` for headers; `<td>` for data cells.
