# Label

## Purpose
Form label with optional required indicator.

**When to use:** Associate text with inputs/controls.

**When not to use:** Standalone headings (use Heading).

## API (Props)

| Prop | Type |
| --- | --- |
| required? | `boolean` |
| as? | `ElementType` |
| htmlFor? | `string` |
| className? | `string` |
| children? | `ReactNode` |

## Examples

```tsx
<Label htmlFor="email" required>Email</Label>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
