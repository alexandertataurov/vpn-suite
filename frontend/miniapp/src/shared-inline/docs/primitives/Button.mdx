# Button

## Purpose
Primary action control for triggering commands.

**When to use:** Primary and secondary actions, toolbars, dialogs.

**When not to use:** Navigation (use ButtonLink) or inline text links (use link styles).

## API (Props)

| Prop | Type |
| --- | --- |
| variant | `"primary"` | "secondary" | "ghost" | "outline" | "danger" | "link" |
| size | `"sm"` | "md" | "lg" | "icon" |
| kind? | `"default"` | "connect" |
| loading? | `boolean` |
| loadingText? | `string` |
| fullWidth? | `boolean` |
| startIcon? | `ReactNode` |
| endIcon? | `ReactNode` |
| iconOnly? | `boolean` |
| asChild? | `boolean` |
| children? | `ReactNode` |
| nativeProps | `Omit<ButtonHTMLAttributes<HTMLButtonElement>, "children">` |

## Examples

```tsx
<Button variant="primary" size="md">Save</Button>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
