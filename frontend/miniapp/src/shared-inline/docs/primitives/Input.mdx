# Input

## Purpose
Single-line text input with optional label/error via Field.

**When to use:** Short text, email, IDs, search terms.

**When not to use:** Multi-line content (use Textarea).

## API (Props)

| Prop | Type |
| --- | --- |
| label? | `string` |
| error? | `string` |
| nativeProps | `InputHTMLAttributes<HTMLInputElement>` |

## Examples

```tsx
<Input label="Email" placeholder="you@example.com" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
