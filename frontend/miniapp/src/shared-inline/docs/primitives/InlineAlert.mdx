# InlineAlert

## Purpose
Inline alert panel for contextual messaging.

**When to use:** Non-blocking info/warning/error banners.

**When not to use:** Full-page error states.

## API (Props)

| Prop | Type |
| --- | --- |
| variant? | `"info" \\| "warning" \\| "error"` |
| title? | `string` |
| message? | `string` |
| actions? | `ReactNode` |

## Examples

```tsx
<InlineAlert variant="warning" title="Check settings" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
