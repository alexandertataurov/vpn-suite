# TableRow

## Purpose
Table row primitive for consistent row styling and interaction.

**When to use:** Table body rows.

**When not to use:** Non-tabular layouts (use Stack or Grid).

## API (Props)

| Prop | Type |
| --- | --- |
| nativeProps | `HTMLAttributes<HTMLTableRowElement>` |

## Examples

```tsx
<TableRow>
  <TableCell>Value</TableCell>
</TableRow>
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using table row outside a table.

## Accessibility
- Use semantic `<table>` structure and headers.
