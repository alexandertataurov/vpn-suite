# Heading

## Purpose
Heading primitive with tokenized style.

**When to use:** Section titles and page headers.

**When not to use:** Using for body text.

## API (Props)

| Prop | Type |
| --- | --- |
| variant? | `"title"` |
| as? | `ElementType` |
| className? | `string` |
| children? | `ReactNode` |
| nativeProps | `Omit<HTMLAttributes<HTMLHeadingElement>, "children">` |

## Examples

```tsx
<PrimitiveHeading as="h2">Section</PrimitiveHeading>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Ensure semantic elements and labels when needed.
- Keep keyboard navigation predictable.
