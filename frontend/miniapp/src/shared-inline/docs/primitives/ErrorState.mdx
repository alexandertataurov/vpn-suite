# ErrorState

## Purpose
Inline error message with optional retry.

**When to use:** Section-level failures and fetch errors.

**When not to use:** Full-page errors (use PageError).

## API (Props)

| Prop | Type |
| --- | --- |
| title | `string` |
| message? | `string` |
| retry? | `() => void` |
| className? | `string` |
| nativeProps | `Omit<HTMLAttributes<HTMLDivElement>, "title">` |

## Examples

```tsx
<ErrorState title="Failed to load" retry={onRetry} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
