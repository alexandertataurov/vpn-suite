# Badge

## Purpose
Compact status/label chip.

**When to use:** Statuses, counts, short labels.

**When not to use:** Long text or multi-line content.

## API (Props)

| Prop | Type |
| --- | --- |
| variant? | `"neutral"` | "success" | "warning" | "danger" | "info" |
| size? | `"sm"` | "md" |
| asChild? | `boolean` |
| children? | `ReactNode` |
| nativeProps | `HTMLAttributes<HTMLSpanElement>` |

## Examples

```tsx
<PrimitiveBadge variant="success">Healthy</PrimitiveBadge>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
