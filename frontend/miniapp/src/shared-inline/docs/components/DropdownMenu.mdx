# DropdownMenu

## Purpose
Contextual menu for small action sets.

**When to use:** Row actions, compact menus.

**When not to use:** Large form interactions.

## API (Props)

| Prop | Type |
| --- | --- |
| items | `{ label: string; onSelect: () => void }[]` |
| align? | `"start"` | "end" |
| trigger? | `ReactNode` |

## Examples

```tsx
<DropdownMenu items={items} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
