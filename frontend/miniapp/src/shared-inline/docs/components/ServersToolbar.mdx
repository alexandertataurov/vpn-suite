# ServersToolbar

## Purpose
Toolbar for servers list with status and filters.

**When to use:** Servers page only (operator context).

**When not to use:** Generic filtering (use FilterBar).

## API (Props)

| Prop | Type |
| --- | --- |
| dataUpdatedAt? | `number` |
| isFetching | `boolean` |
| onSync | `() => void` |
| connectionState | `"live"` | "degraded" | "offline" |
| liveIntervalSeconds? | `number` |
| children? | `ReactNode` |

## Examples

```tsx
<ServersToolbar isFetching={false} onSync={sync} connectionState="live" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.
