# Modal

## Purpose
Modal dialog for blocking confirmation or forms.

**When to use:** Short forms or confirmations requiring focus.

**When not to use:** Deep workflows; use a page.

## API (Props)

| Prop | Type |
| --- | --- |
| open | `boolean` |
| onClose | `() => void` |
| title? | `string` |
| footer? | `ReactNode` |
| children | `ReactNode` |

## Examples

```tsx
<Modal open onClose={onClose} title="Confirm">...</Modal>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
