# FilterBar

## Purpose
Search + filter toolbar for list pages.

**When to use:** Servers and other list views with common filters.

**When not to use:** Simple lists (use SearchInput + Select directly).

## API (Props)

| Prop | Type |
| --- | --- |
| search | `string` |
| onSearch | `(value: string) => void` |
| statusOptions | `SelectOption[]` |
| statusValue | `string` |
| onStatusChange | `(value: string) => void` |
| sortOptions | `SelectOption[]` |
| sortValue | `string` |
| onSortChange | `(value: string) => void` |
| lastSeenOptions? | `SelectOption[]` |
| lastSeenValue? | `string` |
| onLastSeenChange? | `(value: string) => void` |
| regionOptions? | `SelectOption[]` |
| regionValue? | `string` |
| onRegionChange? | `(value: string) => void` |
| density? | `"compact" \| "normal"` |
| onDensityChange? | `() => void` |

## Examples

```tsx
<FilterBar
  search={query}
  onSearch={setQuery}
  statusOptions={statusOptions}
  statusValue={status}
  onStatusChange={setStatus}
  sortOptions={sortOptions}
  sortValue={sort}
  onSortChange={setSort}
/>
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Duplicating FilterBar layout in pages.

## Accessibility
- Provide `aria-label` for inputs.
- Ensure focus ring visibility.
