# CodeBlock

## Purpose
Preformatted code block with optional actions.

**When to use:** Configs, commands, logs.

**When not to use:** Inline code (use CodeText).

## API (Props)

| Prop | Type |
| --- | --- |
| value | `string` |
| language? | `"ini"` | "conf" | "text" |
| maxHeight? | `number` |
| wrap? | `boolean` |
| actions? | `ReactNode` |

## Examples

```tsx
<CodeBlock value="[Server]" language="ini" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
