# StatusStrip

## Purpose
Compact strip for multi-signal status summaries.

**When to use:** Operator headers or summary bars with multiple signals.

**When not to use:** Single status values (use StatusBadge/HealthBadge).

## API (Props)

Use `StatusStrip` with label/value pairs and optional icons.

## Examples

```tsx
<StatusStrip
  items={[
    { label: "Agents", value: "Healthy" },
    { label: "Alerts", value: "2" },
  ]}
/>
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Typography: `--text-*`

## Anti-patterns
- Overloading with too many items.
- Inline styles or arbitrary spacing.

## Accessibility
- Ensure labels are readable and values have sufficient contrast.
