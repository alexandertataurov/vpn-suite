# Forms

Standard form patterns using Field, Input, Checkbox, Select, FormStack.

## Form stack

Use FormStack for vertical spacing between form fields.

```tsx
import { FormStack, Field, Input, Button } from "@vpn-suite/shared/ui";

<FormStack>
  <Field label="Email">
    <Input type="email" placeholder="you@example.com" />
  </Field>
  <Field label="Password">
    <Input type="password" placeholder="••••••••" />
  </Field>
  <Button>Submit</Button>
</FormStack>
```

## Validation

- Use `error` on Field for inline validation messages
- Use InlineAlert for form-level errors
