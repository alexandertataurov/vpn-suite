# Tables with Actions

Data tables with sorting, selection, and bulk actions.

## Components

- **Table** — Core data table with columns, sorting, density
- **TableContainer** — Standard table container (scroll + chrome)
- **BulkActionsBar** — Shown when rows are selected
- **PrimitiveTableCell** — Truncation, numeric alignment (used internally)

## Pattern

1. Table with selectable rows
2. Selection triggers BulkActionsBar
3. Bulk actions: Delete, Export, etc.
4. Clear selection to dismiss
