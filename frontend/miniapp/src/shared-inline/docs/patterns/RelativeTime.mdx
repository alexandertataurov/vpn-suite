# RelativeTime

## Purpose
Relative time display with tooltip of exact time.

**When to use:** Status updates, last-seen timestamps.

**When not to use:** Legal/audit timestamps (use absolute time).

## API (Props)

| Prop | Type |
| --- | --- |
| date | `Date \\| string` |
| title? | `string` |
| updateInterval? | `number` |
| className? | `string` |

## Examples

```tsx
<RelativeTime date={new Date()} updateInterval={60000} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
