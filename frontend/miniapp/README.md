# Miniapp (Telegram Mini App)

React app for VPN subscription and device management. Base path `/webapp/`.

**Design system:** [architecture guide](src/design-system/docs/design-system-architecture.md), [enforcement checklist](src/design-system/docs/design-system-enforcement-checklist.md), and [system README](src/design-system/README.md). Token aliases: [src/design-system/styles/miniapp-primitives-aliases.css](src/design-system/styles/miniapp-primitives-aliases.css). Aligned with admin Primitives: [admin design-system docs](../admin/src/design-system/docs/design-system.md).

**Layer model:** Tokens → Foundations → Primitives → Components → Patterns → Features.

**Layout and tokens:** See [docs/frontend/miniapp-layout-architecture.md](../../docs/frontend/miniapp-layout-architecture.md).

**Run:** From repo root, `./manage.sh up-core` (or frontend dev server on port 5175).
