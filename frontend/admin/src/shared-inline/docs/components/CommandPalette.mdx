# CommandPalette

## Purpose
Keyboard-driven command palette for quick navigation and actions.

**When to use:** Power-user navigation and action search.

**When not to use:** Simple menus (use DropdownMenu).

## API (Props)

| Prop | Type |
| --- | --- |
| open | `boolean` |
| onClose | `() => void` |
| items | `{ id: string; label: string; keywords?: string; onSelect: () => void }[]` |

## Examples

```tsx
<CommandPalette open onClose={close} items={items} />
```

## Design Tokens Used
- Spacing: `--spacing-*`
- Colors: `--color-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Large ungrouped command lists without search keywords.

## Accessibility
- Keyboard navigation is required (arrow keys, Enter, Escape).
