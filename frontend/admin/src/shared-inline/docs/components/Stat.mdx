# Stat

## Purpose
Small metric display with label/value.

**When to use:** Inline KPI blocks.

**When not to use:** Dense tables (use MetricRow/Table).

## API (Props)

| Prop | Type |
| --- | --- |
| label | `string` |
| value | `ReactNode` |
| help? | `string` |

## Examples

```tsx
<Stat label="Uptime" value="99.9%" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
