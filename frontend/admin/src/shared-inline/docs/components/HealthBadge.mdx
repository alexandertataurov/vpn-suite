# HealthBadge

## Purpose
Status badge for health signals.

**When to use:** System health labels.

**When not to use:** General tags (use Badge).

## API (Props)

| Prop | Type |
| --- | --- |
| status | `"healthy"` | "degraded" | "down" | "unknown" |

## Examples

```tsx
<HealthBadge status="healthy" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.
