# Pagination

## Purpose
Table pagination control with previous/next and page sizes.

**When to use:** Server-side or client-side paged tables.

**When not to use:** Infinite scrolling contexts.

## API (Props)

| Prop | Type |
| --- | --- |
| offset | `number` |
| limit | `number` |
| total | `number` |
| onPage | `(offset: number) => void` |

## Examples

```tsx
<Pagination offset={0} limit={20} total={120} onPage={setOffset} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
