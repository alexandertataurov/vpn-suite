# Icons

Operator UI uses the Lucide icon set. Icons must align to the tokenized size scale and color tokens.

## Sizes

- Small: `--icon-size-sm`
- Medium: `--icon-size-md`
- Large: `--icon-size-lg`

## Usage

- Use `currentColor` so icons inherit text color.
- Always pair icon-only buttons with `aria-label`.

## Do

- Keep icon size consistent within a component.
- Use icons to reinforce labels, not replace them.

## Don't

- Use raw SVG fills or hex colors.
- Use icons as the only affordance for critical actions.
