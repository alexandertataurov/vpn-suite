# Tokens

This design system is token-first. All components consume semantic tokens only.

## Core semantic tokens

- `--color-bg`, `--color-surface`, `--color-surface-2`, `--color-overlay`
- `--color-text`, `--color-text-muted`, `--color-text-tertiary`, `--color-text-inverse`
- `--color-border`, `--color-border-subtle`, `--color-border-strong`
- `--color-accent`, `--color-accent-hover`, `--color-accent-active`, `--color-on-accent`
- `--color-success`, `--color-warning`, `--color-danger`, `--color-info`
- `--color-focus-ring`

## Spacing

4px base scale. Use `--spacing-*` tokens and `--space-*` aliases.

## Typography

Strict sizes only: 12, 13, 14, 16. Use `--text-*` composite tokens.
