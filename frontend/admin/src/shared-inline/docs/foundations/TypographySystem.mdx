# Typography System

Fluid type scale. Use composite `--text-*` tokens.

## Primitives (composition only)

- `--font-size-xs` … `--font-size-5xl`
- `--font-weight-*`, `--line-height-*`, `--letter-spacing-*`
- `--font-sans`, `--font-mono`, `--font-serif`

## Semantic (use in components)

| Token | `Use` |
| ------- | `-----` |
| `--text-h1` … `--text-h6` | `Headings` |
| `--text-body`, `--text-body-lg`, `--text-body-sm` | `Body` |
| `--text-button`, `--text-caption`, `--text-overline` | `UI roles` |

## Usage Contract

- **Use** `--text-h1` … `--text-body` for typography
- **Don't** use raw `font-size` / `font-weight` — compose from tokens
- **Numeric** — use `font-variant-numeric: tabular-nums` for stats/tables
