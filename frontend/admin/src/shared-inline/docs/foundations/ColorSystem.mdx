# Color System

Three-layer color architecture. Use **semantic tokens** in components.

> **Theme switcher:** Use the toolbar above (dark | light | dim) to preview semantic colors in each theme. Semantics adapt to the active theme.

- **Primitives** — Raw ramps (gray, primary, status). Composition only.
- **Semantics** — Background, text, border, interactive. Use these.
- **Usage Contracts** — When to use, anti-patterns, component mapping.

Use the sidebar to navigate: Primitives → Semantics → Usage Contracts.
