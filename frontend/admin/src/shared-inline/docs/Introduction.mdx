# VPN Suite Design System

Enterprise design system for the VPN Suite platform. Use this documentation to build consistent, accessible interfaces.

## Structure

- **Foundations** — Primitives, semantics, usage contracts (color, typography, spacing, elevation, radius, motion, governance)
- **Components** — Reusable UI building blocks
- **Patterns** — Forms, tables, empty states, loading states
- **Admin** — Admin-specific components and patterns

## Getting started

1. Use **semantic tokens** only (no primitives, no raw hex) — see Foundations/Governance
2. Compose from shared components in `@/design-system`
3. Follow accessibility guidelines (keyboard, focus, ARIA)
