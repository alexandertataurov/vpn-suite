# Inline

## Purpose
Inline horizontal layout using stack tokens.

**When to use:** Button rows, label-value pairs.

**When not to use:** Complex grid layouts.

## API (Props)

| Prop | Type |
| --- | --- |
| gap? | `number` | string |
| wrap? | `boolean` |
| align? | `"start"` | "center" | "end" | "stretch" |
| children | `ReactNode` |
| className? | `string` |

## Examples

```tsx
<Inline gap="2">...</Inline>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Ensure semantic elements and labels when needed.
- Keep keyboard navigation predictable.
