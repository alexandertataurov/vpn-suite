# Select

## Purpose
Native select with options list.

**When to use:** Small option sets and system filters.

**When not to use:** Large datasets or async search (use a searchable dropdown).

## API (Props)

| Prop | Type |
| --- | --- |
| options | `{ value: string; label: string }[]` |
| value | `string` |
| onChange | `(value: string) => void` |
| label? | `string` |
| error? | `string` |
| id? | `string` |
| nativeProps | `Omit<SelectHTMLAttributes<HTMLSelectElement>, "value"` | "onChange"> |

## Examples

```tsx
<Select options={opts} value="all" onChange={setValue} />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
