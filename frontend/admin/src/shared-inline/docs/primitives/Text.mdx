# Text

## Purpose
Text primitive with tokenized variants.

**When to use:** Body, label, caption, code text.

**When not to use:** Custom font sizing in components.

## API (Props)

| Prop | Type |
| --- | --- |
| variant? | `"body"` | "label" | "caption" | "code" | "muted" |
| as? | `ElementType` |
| className? | `string` |
| children? | `ReactNode` |
| nativeProps | `Omit<HTMLAttributes<HTMLElement>, "children">` |

## Examples

```tsx
<PrimitiveText variant="body">Body text</PrimitiveText>
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Ensure semantic elements and labels when needed.
- Keep keyboard navigation predictable.
