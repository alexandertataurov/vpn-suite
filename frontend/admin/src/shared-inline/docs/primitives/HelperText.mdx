# HelperText

## Purpose
Supporting text for form inputs.

**When to use:** Clarify input format or constraints.

**When not to use:** Validation errors (use InlineError).

## API (Props)

| Prop | Type |
| --- | --- |
| children | `ReactNode` |
| nativeProps | `HTMLAttributes<HTMLParagraphElement>` |

## Examples

```tsx
<HelperText>Must be at least 8 characters.</HelperText>
```

## Design Tokens Used
- Typography: `--text-*`
- Colors: `--color-text-muted`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Redundant or verbose helper copy.

## Accessibility
- Associate helper text via `aria-describedby` on inputs.
