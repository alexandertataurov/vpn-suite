# EmptyState

## Purpose
No-data placeholder with optional actions.

**When to use:** Empty lists, filtered results with no rows.

**When not to use:** Error conditions (use ErrorState).

## API (Props)

| Prop | Type |
| --- | --- |
| icon? | `ReactNode` |
| title | `string` |
| description? | `string` |
| actions? | `ReactNode` |
| className? | `string` |

## Examples

```tsx
<EmptyState title="No servers" description="Add your first node" />
```

## Design Tokens Used
- Typography: `--text-*`
- Spacing: `--spacing-*`
- Colors: `--color-*`
- Radius: `--radius-*`

## Anti-patterns
- Inline styles or arbitrary pixel values.
- Using component for unrelated semantics.

## Accessibility
- Provide labels and aria attributes as needed.
- Ensure focus visibility and keyboard access.
